# Guía de contribución

Este repositorio utiliza Issues y Pull Requests para organizar y dar seguimiento a las actividades de la PPS.

## Títulos de Issues

Se recomienda utilizar el formato:

`[Área] Descripción breve`

Ejemplos:

* `[Auditoría] sitio.com - Página de inscripción`
* `[Subtitulado] Matemática 1 - Primera clase`
* `[Documentos] Presentación accesible`
* `[Wiki] Guía de WAVE`
* `[Proyecto] Actualizar README`

## Títulos de Pull Requests

Se recomienda utilizar el mismo título del Issue asociado o una variación mínima que describa claramente los cambios realizados.

Ejemplo:

Issue:

`[Wiki] Crear guía de WAVE`

Pull Request:

`[Wiki] Crear guía de WAVE`

## Áreas sugeridas

- Auditoría
- Subtitulado
- Documentos
- Wiki
- Proyecto
